# wtpractice
В рамках практикума студенты разрабатывают небольшое Web-приложение с использованием среды интеграции Spring и преобразователя реляционных данных в объектные Hibernate. Для разработки используются технологии Java EE, среда Eclipse, инструменты разработки и выполнения тестов TestNG и Selenium.
